# WildTransfer

WildTransfer is a fast code-based file sharing app from Ocean and Wild Studios.

## Current scope
- Send up to 10 files per transfer
- Receive files by 6-character code
- Upload progress, cancel upload, and local activity history
- Backend API compatibility: `/api/wild-transfer/*`

## Run
Serve this file from your backend static route:
- `/wild-transfer` -> `index.html`
